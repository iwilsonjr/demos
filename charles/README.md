# [Charles/Delta TWP](http://www.thewilsonproject.com/)

The "Charles" theme is the fourth version of The Wilson Project site (2001-), named after the bicentennial of Charles Darwin's birth in 2009.

* Author : [Ivan Wilson](http://www.thewilsonproject.com/) // [@iwilsonjr](http://twitter.com/iwilsonjr)

This custom theme is a based on HTML5 Blank theme of Todd Motto and et. al. (see below). Various portions of the original theme have been altered or remove according to need/design.

* Project: [github.com/toddmotto/html5blank](https://github.com/toddmotto/html5blank)
* Website: [html5blank.com](http://html5blank.com)
* Twitter: [@html5blank](http://twitter.com/html5blank)
* Author : [Todd Motto](http://toddmotto.com) // [@toddmotto](http://twitter.com/toddmotto)
